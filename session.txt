Git insallation

Then open gitbash from the specific folder by right clicking and choosing open gitbash
Inside gitbash
-->Git init
-->git branch
-->git status(to check if any new file is created)
-->git add (file name)=>staging --- for adding a single file
OR
-->git add .   ---> for adding all the files present in the folder
-->(additional command )rm -rf file name --->to delete the file from the local system
-->git reset file name --->to make a file unstaging so that it will not commit
